# Data Science Project: Healthcare & E-Commerce Analysis

This repository contains two data science projects:

1. Heart Disease Dataset Analysis
2. E-Commerce Sales Analysis

The project demonstrates the complete data science workflow:
- Data preprocessing
- Exploratory data analysis
- SQL queries
- Data visualization
- Dashboard creation
- Insight generation

## Datasets

### Heart Disease Dataset
Used to analyze medical indicators associated with cardiovascular diseases.

### Brazilian E-Commerce Dataset
Real marketplace dataset containing orders, customers, sellers, and product data.

## Tools Used

Python  
Pandas  
Matplotlib  
Seaborn  
SQL  
Power BI / Tableau  

## Project Structure

datasets – raw datasets  
notebooks – analysis notebooks  
sql_queries – SQL analysis  
dashboards – visual dashboards  
docs – dataset documentation
